## Description:
Styling using Tailwind CSS framwork.

### Build with
- [Javascript](https://www.javascript.com/)
- [Tailwind](https://tailwindcss.com/)
- [Vite](https://vite.dev/)
  
---

## Getting Started

### Clone the repo:

```
git clone https://github.com/annechh/fed2-js2-css-framworks.git
````

#### Install the dependencies:
````
npm install
````

#### To run the app
````
npm run dev
````
